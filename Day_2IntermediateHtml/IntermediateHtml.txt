Lists :
1] Unordered list : <ul>
                        <li>Milk</li> // these are list items
                        <li>Eggs</li> // if we want to add any content to our list we need to use these list items tag
                    </ul>
                    Unordered list when executed display a bullet point infront of the item 

2] Ordered List : <ol>
                    <li>Milk</li>
                    <li>Eggs</li>
                  </ol>
                  // here it will display a number before the element
